# 🍛 Indoishii – Restaurant Ordering System with LINE Integration

A modern restaurant ordering system with LINE integration, designed for seamless food ordering and customer engagement.
This system allows users to browse menu items, select dishes, and place orders directly via LINE.

---

## 🚀 Key Features

* 📱 Mobile-friendly UI for easy ordering
* 🍛 Dynamic Menu Display (Curry, Items, Categories)
* 🛒 Add to Cart functionality
* 💬 LINE Integration for order placement
* 📄 Item Details & Custom Selection
* ⚡ Fast and simple user experience

---

## 🏢 Use Case

Developed for a restaurant business to:

* Accept orders digitally
* Reduce manual ordering process
* Integrate with LINE (popular in Japan 🇯🇵)
* Improve customer experience

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript, Bootstrap
* **Backend:** PHP
* **Database:** MySQL
* **Integration:** LINE Messaging API

---

## ⚙️ Installation & Setup

```bash
# Move project to XAMPP htdocs
C:\xampp\htdocs\indoishii

# Start Apache & MySQL from XAMPP

# Open in browser
http://localhost/indoishii
```
